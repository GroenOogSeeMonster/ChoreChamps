# ChoreChamps

ChoreChamps is a full-stack web application designed to help families manage household chores for children. It features an admin console for parents to assign chores and a kids console for children to view and complete their assigned tasks.

## Features

- User authentication (login for both parents and children)
- Admin console for parents to:
  - View all registered users
  - Assign chores to specific children
  - Add new chores to the system
- Kids console for children to:
  - View their assigned chores
  - Mark chores as completed
- RESTful API backend
- React-based frontend
- PostgreSQL database for data persistence
- Dockerized application for easy deployment

## Tech Stack

- Backend: Python, Flask, SQLAlchemy
- Frontend: React, React Router, Axios
- Database: PostgreSQL
- Containerization: Docker, Docker Compose
